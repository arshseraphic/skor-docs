The SKOR Modular API reference is generated from the OpenAPI specification.

Use the **Modular API Reference** section in the sidebar to browse webhook and tenant endpoints. Generate a shared token with `/authenticate/token` in the **Authentication** section, then use `Authorization: Bearer <token>` when calling protected endpoints.

## Base URLs

| Environment | Base URL |
| --- | --- |
| Live | `https://api.skortorent.com/api/v1` |
| Sandbox | `https://dev-api.skortorent.com/api/v1` |
