The SKOR Modular API reference is generated from the OpenAPI specification.

Use the **API Reference** section in the sidebar to browse token, webhook, and tenant endpoints. Generate a token with `/keys/generate-token` first, then use `Authorization: Bearer <token>` when calling protected endpoints.

## Base URLs

| Environment | Base URL |
| --- | --- |
| Live | `https://api.skortorent.com/api/v1` |
| Sandbox | `https://dev-api.skortorent.com/api/v1` |
