## Authentication

Most SKOR API endpoints require a bearer token. First, use your `client_id` and `client_secret` to request a token with HTTP Basic Authentication. Then send that token in the `Authorization` header for protected APIs.

```bash
Authorization: Bearer <token>
```

## Base URL

The OpenAPI reference currently uses the development API server:

```text
https://dev-api.skortorent.com/api/v1
```

Use the production API base URL when your credentials are issued for production.
