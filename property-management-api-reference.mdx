The SKOR Property Management API reference is generated from the OpenAPI specification.

Use this section to browse token, workspace, and property endpoints. Generate a token with `/auth/token` by sending `client_id` and `client_secret` through HTTP Basic Auth. The token expires after one hour. Send it as `Authorization: Bearer <token>` when calling protected endpoints.

<Warning>
  This API does not accept Modular API keys, Modular API tokens, or regular application tokens.
</Warning>
